# 🚀 快速启动指南

## 一分钟启动游戏

### 方法一：使用启动脚本（推荐）

```bash
# 1. 给脚本添加执行权限
chmod +x start-game.sh

# 2. 运行启动脚本
./start-game.sh
```

### 方法二：手动启动

```bash
# 1. 安装依赖
npm install

# 2. 启动游戏
npm run dev
```

## 🎮 游戏说明

### 游戏目标
帮助儿童练习字母输入，熟悉键盘布局

### 游戏规则
1. 地鼠随机从洞中探出头，举着字母牌子
2. 快速按下键盘上对应的字母键来"打击"地鼠
3. 正确击中得分，错过或按错会扣生命值
4. 在时间结束前获得尽可能高的分数

### 难度选择
- **🐌 简单**：慢速度，适合初学者
- **🐰 中等**：中等速度，有一定挑战
- **🚀 困难**：快速度，挑战你的极限

## 🔧 环境要求

- Node.js 16+ 
- 现代浏览器（Chrome、Firefox、Safari、Edge）
